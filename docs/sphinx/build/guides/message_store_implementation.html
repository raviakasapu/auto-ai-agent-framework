Message Store Implementation Guide

Overview

Implementations create message stores in their preferred format (database, file system, etc.) and the framework reads messages directly from the store. This separates storage concerns from framework logic.

Architecture

┌─────────────────────────────────────────────────────────────┐
│  IMPLEMENTATION LAYER                                       │
│                                                             │
│  ┌──────────────┐    ┌──────────────────┐                 │
│  │ Your Store   │───▶│ BaseMessageStore │                 │
│  │ (DB/File/...)│    │ Implementation   │                 │
│  └──────────────┘    └──────────────────┘                 │
│         │                      │                            │
│         │                      │ location reference         │
│         │                      ▼                            │
│         │            ┌──────────────────────┐              │
│         │            │ MessageStoreMemory   │              │
│         │            │ (Framework)          │              │
│         │            └──────────────────────┘              │
│         │                      │                            │
│         └──────────────────────┼──────────────────────────┘
│                                │                            │
└────────────────────────────────┼────────────────────────────┘
                                 │
                                 ▼
┌─────────────────────────────────────────────────────────────┐
│  FRAMEWORK LAYER                                            │
│                                                             │
│  ┌──────────────────────┐                                  │
│  │ ManagerAgent/Agent   │                                  │
│  │ memory.get_history() │───▶ Reads from MessageStore     │
│  └──────────────────────┘                                  │
└─────────────────────────────────────────────────────────────┘

Step 1: Implement BaseMessageStore

from agent_framework.base import BaseMessageStore
from typing import List, Dict, Any, Optional

class YourMessageStore(BaseMessageStore):
    """Your implementation of the message store interface."""
    
    def __init__(self, your_store_connection):
        """Initialize with your store (database, file, etc.)."""
        self.store = your_store_connection
    
    def get_conversation_messages(
        self,
        location: str,
        limit: Optional[int] = None
    ) -> List[Dict[str, Any]]:
        """
        Return conversation messages (user_message, assistant_message).
        
        Location is your identifier (job_id, namespace, etc.).
        """
        # Query your store for conversation messages
        messages = self.store.query(
            location=location,
            types=["user_message", "assistant_message"],
            limit=limit
        )
        # Ensure messages are in framework format
        return [
            {
                "type": msg.type,
                "content": msg.content,
                "timestamp": msg.timestamp,
                "turn_id": msg.turn_id
            }
            for msg in messages
        ]
    
    def get_agent_messages(
        self,
        location: str,
        agent_key: str,
        limit: Optional[int] = None
    ) -> List[Dict[str, Any]]:
        """Return execution traces for a specific agent."""
        messages = self.store.query(
            location=location,
            agent_key=agent_key,
            types=["task", "action", "observation", "error", "final", "delegation"],
            limit=limit
        )
        return [
            {
                "type": msg.type,
                "content": msg.content,
                "timestamp": msg.timestamp,
                "turn_id": msg.turn_id,
                "tool": getattr(msg, "tool", None),
                "args": getattr(msg, "args", None)
            }
            for msg in messages
        ]
    
    def get_global_messages(
        self,
        location: str,
        limit: Optional[int] = None
    ) -> List[Dict[str, Any]]:
        """Return global/broadcast messages."""
        messages = self.store.query(
            location=location,
            types=["global_observation", "synthesis"],
            limit=limit
        )
        return [
            {
                "type": msg.type,
                "content": msg.content,
                "timestamp": msg.timestamp,
                "turn_id": msg.turn_id,
                "from_worker": getattr(msg, "from_worker", None),
                "from_manager": getattr(msg, "from_manager", None),
                "summary": getattr(msg, "summary", None)
            }
            for msg in messages
        ]
    
    def get_team_messages(
        self,
        location: str,
        agent_keys: List[str],
        limit: Optional[int] = None
    ) -> List[Dict[str, Any]]:
        """Return messages from multiple agents."""
        messages = self.store.query(
            location=location,
            agent_keys=agent_keys,
            limit=limit
        )
        return [
            {
                "type": msg.type,
                "content": msg.content,
                "timestamp": msg.timestamp,
                "turn_id": msg.turn_id,
                "agent_key": msg.agent_key
            }
            for msg in messages
        ]

Step 2: Prepare Your Store Structure

Organize messages in your store however you prefer. The framework only cares about the interface.

Example: SQLite Database Schema

CREATE TABLE messages (
    id INTEGER PRIMARY KEY,
    location TEXT NOT NULL,
    agent_key TEXT,
    type TEXT NOT NULL,
    content TEXT NOT NULL,
    timestamp REAL,
    turn_id TEXT,
    tool TEXT,
    args TEXT,  -- JSON
    from_worker TEXT,
    from_manager TEXT,
    summary TEXT
);

CREATE INDEX idx_location_type ON messages(location, type);
CREATE INDEX idx_location_agent ON messages(location, agent_key);

Step 3: Use MessageStoreMemory

from agent_framework.components.message_store_memory import MessageStoreMemory
from agent_framework.core.manager_v2 import ManagerAgent

# Create your store implementation
message_store = YourMessageStore(your_store_connection)

# Create memory that reads from your store
location = "job_123"  # Your location identifier
memory = MessageStoreMemory(
    message_store=message_store,
    location=location,
    agent_key="orchestrator"
)

# Create agent - framework reads from your store
agent = ManagerAgent(
    planner=planner,
    memory=memory,  # Framework will read via memory.get_history()
    workers=workers,
    policies=policies
)

# Run agent - framework reads messages from your store
result = await agent.run(task="List tables", progress_handler=handler)

Step 4: Writing Messages to Your Store

The framework reads from the store, but doesn’t write. You write messages to your store:

from agent_framework.constants import USER_MESSAGE, ASSISTANT_MESSAGE
from agent_framework.utils.message_builder import (
    create_user_message,
    create_assistant_message
)

# When user sends message
user_msg = create_user_message(
    content=user_input,
    timestamp=time.time(),
    turn_id=f"turn_{turn_number}"
)
your_store.write_message(location="job_123", message=user_msg)

# When assistant completes
assistant_msg = create_assistant_message(
    content=result_summary,
    timestamp=time.time(),
    turn_id=f"turn_{turn_number}"
)
your_store.write_message(location="job_123", message=assistant_msg)

from agent_framework.components.message_store_memory import MessageStoreMemory
from agent_framework.core.manager_v2 import ManagerAgent
from agent_framework.constants import USER_MESSAGE, ASSISTANT_MESSAGE
from agent_framework.utils.message_builder import (
    create_user_message,
    create_assistant_message
)

class DatabaseMessageStore(BaseMessageStore):
    def __init__(self, db):
        self.db = db
    
    def get_conversation_messages(self, location, limit=None):
        # Your implementation
        pass
    
    # ... implement other methods

# In your API endpoint
@app.post("/run")
async def run_agent(req: RunRequest):
    job_id = req.job_id
    
    # Create/store user message
    user_msg = create_user_message(content=req.task, timestamp=time.time())
    your_store.write_message(job_id, user_msg)
    
    # Create memory that reads from your store
    message_store = DatabaseMessageStore(your_database)
    memory = MessageStoreMemory(
        message_store=message_store,
        location=job_id,
        agent_key="orchestrator"
    )
    
    # Create and run agent
    agent = ManagerAgent(planner=planner, memory=memory, workers=workers, policies=policies)
    result = await agent.run(task=req.task, progress_handler=handler)
    
    # Store assistant message
    summary = result.get("human_readable_summary") or str(result)
    assistant_msg = create_assistant_message(content=summary, timestamp=time.time())
    your_store.write_message(job_id, assistant_msg)
    
    return result

Key Points

  1. Framework reads, you write: Framework only reads via get_history(). You write messages to your store.

  2. Location is yours: You decide what location represents (job_id, namespace, path, etc.).

  3. Format must match: Messages must have type and content fields matching framework constants.

  4. Interface contract: Implement all 4 methods of BaseMessageStore.

  5. Organize freely: Your store structure is your choice - framework only sees the interface.

Runtime Messages

The framework will call memory.add() during execution, but MessageStoreMemory.add() is a no-op. If you need runtime message storage, you can:

  1. Option A: Store them yourself when framework calls memory.add() (hook into your store)

  2. Option B: Use SharedInMemoryMemory for runtime, MessageStoreMemory for initial history

  3. Option C: Hybrid approach - read from store for initial history, use in-memory for runtime
