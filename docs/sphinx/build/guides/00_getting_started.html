Getting Started (Complete Guide)ïƒ

Welcome to the AI Agent Framework! This guide will get you from zero to a working agent in 10 minutes.

Prerequisitesïƒ

Before you begin, make sure you have:

  • Python 3.11 or higher

  • pip (Python package installer)

  • A terminal/command prompt

  • (Optional) A code editor like VS Code

Quick Checkïƒ

Open your terminal and verify Python is installed:

python3 --version

You should see something like Python 3.11.x or higher.

Installationïƒ

Step 1: Clone or Navigate to the Repositoryïƒ

Clone the repo (or pull the latest changes) and open it:

git clone https://github.com/akasa-ai/AI_Agent_Framework.git
cd AI_Agent_Framework

Step 2: Create Virtual Environmentïƒ

Create an isolated Python environment for the project:

python3 -m venv venv

This creates a venv/ directory with a clean Python installation.

Step 3: Activate Virtual Environmentïƒ

On macOS/Linux:

source venv/bin/activate

On Windows:

venv\Scripts\activate

You should see (venv) appear in your terminal prompt.

Step 4: Install Dependenciesïƒ

Install the reusable library plus repo tooling:

pip install -e ./agent-framework-pypi
pip install -r requirements.txt

The editable install exposes the agent_framework package locally while requirements.txt covers FastAPI, Uvicorn, Pydantic, Sphinx, and tooling for the docs server.

Verify Installationïƒ

Test that everything is working:

Key Conceptsïƒ

Before diving deeper, understand these core concepts:

1. Agent

An autonomous entity that can use tools to accomplish tasks.

2. Tool

A specific capability (e.g., search, calculate, read file).

3. Planner

Decides which tool to use and when. Can be:

  • StaticPlanner - Always uses the same tool

  • LLMRouterPlanner - Uses LLM to decide based on user input

4. Gateway

Interface to LLMs (OpenAI, Anthropic, or mock for testing).

5. Memory

Stores conversation history and context.
