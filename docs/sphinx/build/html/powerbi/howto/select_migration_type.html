Select a Migration Type

Purpose

Choose which source→target flow the app uses.

Steps

  1. Set VITE_MIGRATION_TYPE in environment: - tableau-to-powerbi - powerbi-to-tableau - cognos-to-powerbi - looker-to-powerbi

  2. Restart the frontend to apply routing changes.

  3. Navigate to the flow page (e.g., /migrations/tableau-to-powerbi).

Tips

  • Start with a sample dataset to validate the pipeline end‑to‑end.
