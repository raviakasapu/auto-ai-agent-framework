Using the Docs Server
=====================

1. Generate an Agent Manifest.

   .. code-block:: bash

      python generate_manifest.py \
        --config configs/agents/research_assistant.yaml \
        --output docs/agent_manifest.json

2. Start the docs server.

   .. code-block:: bash

      AGENT_MANIFEST_PATH=docs/agent_manifest.json \
      AGENT_DOCS_DIR=docs/sphinx/build/html \
      uvicorn docs_server.main:app --reload

3. Browse:

   - ``/`` Manifest view (tools, params, returns)
   - ``/pages`` Markdown docs under ``docs/``
   - ``/reference`` Sphinx site if built to ``docs/sphinx/build/html``
   - ``/examples`` Example scripts and configs with raw views

Tips
----

- Specify a port: add ``--port 8002``
- Run from repo root; set ``--app-dir .`` if imports fail.
