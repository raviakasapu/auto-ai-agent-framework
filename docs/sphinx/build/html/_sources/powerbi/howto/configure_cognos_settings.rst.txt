Configure Cognos Settings
=========================

Purpose
-------
Enable Cognos → Power BI migrations by configuring server URL and session API key.

Steps
-----
1. Set environment: ``VITE_MIGRATION_TYPE=cognos-to-powerbi``
2. In the app Settings → Cognos Settings:
   - Cognos Server URL (e.g., https://cognos.example.com)
   - Session API Key (securely stored via backend)
3. Backend implements endpoints to persist and validate settings.
4. Start the migration flow from the Cognos → Power BI page.

Notes
-----
- Do not store secrets in source control.
- Validate connection before running a full migration.
