Migration Utilities
===================

Overview
--------
Helper functions used across flows (file handling, mapping, status reporting).

Usage Tips
----------
- Keep utils pure and stateless; pass dependencies explicitly.
- Add input validation and clear exceptions for operator errors.
- Write small tests for mapping/transformation helpers.

Where to Look
-------------
- Project ``docs/MIGRATION-UTILS.md`` describes examples and usage patterns.
