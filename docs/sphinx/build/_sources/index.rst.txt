Architecture Overview
---------------------

**agent-framework-pypi/src/agent_framework/**
   Core reusable library:

   - ``Agent`` / ``ManagerAgent`` and ``EventBus`` for orchestration
   - Base classes for tools, planners, memory, prompt managers, and gateways
   - Policy system plus ``progress_filters`` for UI streaming
   - OpenAI / Google AI gateways, utility tools, and decorators

**bi_tools/**
   Power BI reference implementation:

   - ``DataModelService`` plus domain-specific tools (columns, measures, relationships)
   - Examples showing how to extend the generic framework in a vertical
